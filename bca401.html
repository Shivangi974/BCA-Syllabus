C++ Vs JAVA, JAVA and internet and WWW, JAVA support systems, JAVA environment. Defining a class, adding variables and methods, creating objects, accessing data members, constructors, methods overloading, static members, nesting of methods. Creating threads, extending the threads class, stopping and blocking a thread, life cycle of a thread, using thread methods, thread exceptions, thread priority, synchronization, implementing the runnable interface.

Local and remote applets, applets Vs applications, writing applets, applets life cycle, creating and executable applet, designing a web page, applet tag adding applet to HTML file, running the applet, passing parameters to applets, aligning the display, DTML tags & applets, getting input from the user.
